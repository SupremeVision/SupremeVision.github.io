使用GitHub和Hexo搭建个人博客

1.安装Git
2.安装Node.js
3.进入GitHub，注册登录。
以上步骤没什么难点，而且网上教程很多，不做介绍。

接下来进入重点：

##GitHub相关操作
1.找到New repository按钮，点击。
2.Repository name必须命名为：账户名.github.io（其他不行），并且勾选Initialize this repository with a README。完成创建。

##Git相关操作
1.输入cd ~/.ssh检查.ssh的文件夹
2.输入ssh -keygen -t rsa -C “your e-mail”回车,回车，输入y,回车,完成ssh key的创建。
3.找到id_rsa.pub文件，将内容复制，在GitHub的settings，选择SSH and GPG keys，点击new SSH key，将内容粘贴提交。
4.在Git内先cd ~/.ssh，然后输入ssh -T git@github.com，如果看到Hi后面是你的用户名，就说明成功了。

##Hexo相关操作
1.在我的电脑，任选磁盘内建立一个文件夹，如Blog文件夹,在文件夹内右键，选择Get Base Here，进入Git界面。
2.输入npm install -g hexo-cli，安装Hexo。输入hexo -v，检查是否安装成功。
3.输入hexo init，初始化文件夹，结尾显示Start blogging with Hexo则成功。
4.输入npm install，安装所需要的组件。
完成后，指定文件夹下会有目录。
5.新建文章，输入 hexo nwe “Hello World”
6.输入hexo g d，生成并部署
7.可选（输入hexo server，然后在浏览器输入http://localhost:4000可以查看网站。这个只是在本地访问）。
8.在文件夹中找到_config.yml文件，打开，在底部找到

# Deployment
## Docs: https://hexo.io/docs/deployment.html
deploy:

在下面添加代码，格式如下：

type: git
repo: git@github.com:Vi/Vi.github.io.git(这个设置为自己的SSH)
branch: master

注意：冒号后面加空格

9.在Git内输入hexo g d，更新即可。

##如何更新文章：##

1.将md文件添加到\source_posts文件夹内。
2.在/hexo内右键打开Git，输入hexo clean，回车，输入hexo g d。
